Conceals the squad if they see no visible enemies at the beginning of your turn.

You can configure the concealment mode in 3 ways: Squad, Individual, and None. (default squad)

- Squad concealment sets the whole squad as concealed
- Individual works as if everyone used the conceal ability

You can configure the number of turns without seeing an enemy to trigger concealment. (default 1)

This mod is compatible with all other mods.

Warning: Any saved games with this mod enabled must be loaded with this mod enabled.
